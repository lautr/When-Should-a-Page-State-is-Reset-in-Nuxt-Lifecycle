## Log types

- `[INFO] vue: [page] / [hook]`
  - Vue.js lifecycle hook
  - `created`, `beforeUpdate`, `destroyed`, etc.
- `[INFO] vue-router: [page] / [guard]`
  - vue-router navigation guard
  - `beforeRouteEnter`, `beforeRouteUpdate`, `beforeRouteLeave`
- `[INFO] nuxt: [page] / [method]`
  - Nuxt.js method
  - `asyncData`, `fetch`
